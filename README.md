## 博客管理系统
❤包括 前台页面和后台管理系统  

使用HTML,CSS,bootstrap构建静态页面

使用java的SSH2框架，结合MySQL和tomacat搭建后台，以及前后台的交互  

实现了前台游客浏览文章，发表评论，给博主留言等功能  

后台实现的主要功能有使用ckeditor发表文章，查看留言和评论，上传图片等功能
